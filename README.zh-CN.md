# Claude Code 环境管理器

## 概述
Claude Code 环境管理器是一个图形用户界面（GUI）应用程序，用于配置和启动带有特定设置的 Claude 环境。

## 特性
- 管理不同服务的 API 密钥
- 为 API 端点配置自定义基础 URL
- 选择并指定要使用的模型
- 选择环境的工作目录
- 预览配置的环境变量
- 使用配置的设置启动 Claude 环境

## 使用方法
1. **API 密钥管理**：输入或选择用于身份验证的 API 密钥
   - 名称以 `_API_KEY` 结尾的环境变量会加载到 API 密钥下拉菜单中
2. **基础 URL 配置**：设置 API 端点的自定义基础 URL
   - 名称以 `_BASE_URL` 结尾的环境变量会加载到基础 URL 下拉菜单中
3. **模型选择**：为你的操作选择或输入模型
4. **文件夹选择**：使用浏览按钮选择工作目录
5. **环境预览**：在启动前查看环境变量
6. **启动**：使用你的配置启动 Claude 环境

## 运行环境
- 需要 .NET 8.0 或更高版本
- 支持 Windows、Linux 和 macOS

## 快速入门
1. 克隆仓库
2. 使用 Visual Studio 或 .NET CLI（`dotnet build`）构建解决方案
3. 运行应用程序并配置你的设置
4. 启动一个 Claude 环境
