# Claude Code 环境变量管理器
## 功能
- 配置 ANTHROPIC_API_KEY
- 配置 ANTHROPIC_BASE_URL
- 配置 ANTHROPIC_MODEL
- 记住用户输入框输入的内容，下次打开还原数据
- 选择启动的文件夹
- 点击启动，会在目标文件夹启动执行 claude 命令

### API KEY 配置
遍历当前进程的环境变量，找到 _API_KEY 结尾的变量，除了 ANTHROPIC_API_KEY，填入 combobox 里面

除了可以用 combobox 配置，也可以用输入框配置

### BASE_URL 配置
遍历当前进程的环境变量，找到 _BASE_URL 结尾的变量，除了 ANTHROPIC_BASE_URL，填入 combobox 里面

除了可以用 combobox 配置，也可以用输入框配置

### MODEL 配置
combobox 内置：Pro/moonshotai/Kimi-K2-Instruct、moonshotai/Kimi-K2-Instruct、kimi-latest、kimi-k2-0711-preview

除了可以用 combobox 配置，也可以用输入框配置

## 技术要求
- AOT 兼容
- 通过 Avalonia 支持 Windows 和 Linux
- 用 VB XML literals 实现序列化和反序列化以确保能 AOT 兼容
- Option Strict On 并且禁用反射以确保运行时能 AOT 兼容
- 不要安装新的依赖，不要用基于反射的技术存储配置文件
